This folder contains library databases with generic, reusable objects that can be referenced
by client libraries and applications.

StdLib.accde - A collection of general purpose VBA and Windows API-based objects that perform 
routine and repetitive tasks in one line of self documenting code. 

LibOneNote.accde - A collection of low-level OneNote API-based functions that work with 
OneNote objects at the XML document and node levels.

LibOCR.accde - A collection of generic objects used by the client applications to implement 
core functionality such as text parsing, data storage, and database, file and process 
management.

LibAIOOcr.accde - Combines all libraries into a single referencable database.
