This folder contains library databases with generic, reusable objects that can be referenced
by client libraries and applications.

StdLib.accde - A collection of low-level Windows API and general purpose VBA objects that perform 
routine and repetitive tasks in one line of self documenting code. 

LibOCR.accde - A collection of generic objects used by the client applications, such as 
text parsing, and database and process management.

LibOneNote.accde - A collection of low-level OneNote API-based functions.
